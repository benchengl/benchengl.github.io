Moments

For a random variable \(X\) taking values in \(\mathbb{R}^m\), its \(r=(r_1,\cdots, r_m)\)th-moments is defined as \[\alpha_{r_1,\cdots, r_m}=E[X_1^{r_1}\cdots X_m^{r_m}]\] The moment generating function (MGF) is \[M_X(u)=E[\textrm{exp}(u^TX)], u\in \mathbb{R}^m\]

And thus we can compute the moments using MGF as follows: \[\alpha_{r_1,\cdots, r_m}=\frac{\partial^{r_1+\cdots+r_m}M_X(u)}{\partial u_1^{r_1}\cdots\partial u^m^{r_m}}\] Also define the cumulant generating function (CGF) as \[K_X(u)=\log M_X(u)\] and let the following \(\kappa_{r_1,\cdots, r_m}\) be the cumulants: \[\kappa{r_1,\cdots, r_m}=\frac{\partial^{r_1+\cdots+r_m}K_X(u)}{\partial u_1^{r_1}\cdots\partial u^m^{r_m}}|_{u=0}\]
