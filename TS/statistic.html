Statistic

A statistic is a function of observations \(X_1,\cdots, X_n\).

Example: \(h(X_1,\cdots, X_n)=X_1+\cdots+X_n\) is a statistic.

A statistic \(T(X)\) is sufficient if there eixsts nonnegative \(g_{\theta},h\) that \[f(x|\theta)=g_{\theta}(T(x))h(x)\] which is called the factorization criterion. A sufficient statistic is minimal if for any other sufficient statistic \(S\), \(T\) is a function of \(S\).

An estimand is a function \(g(\theta)\) of parameter \(\theta\). An estimator is used to estimate the estimand, \(\tilde{g}(\theta)\). Loss function is a bivariate function \(L(g,\tilde{g})\) used to calculate the difference between estimator and estimand. And risk is defined as the expectation of the loss function.
